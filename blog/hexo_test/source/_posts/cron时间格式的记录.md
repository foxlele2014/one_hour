1. 凌晨3点每天

┌───────────── 分钟 (0-59)
│ ┌───────────── 小时 (0-23)
│ │ ┌───────────── 日期 (1-31)
│ │ │ ┌───────────── 月份 (1-12)
│ │ │ │ ┌───────────── 星期 (0-6) (周日=0)
│ │ │ │ │
0 3 * * *

第一个 0 表示分钟数为0
第二个 3 表示小时数为3
第三个 * 表示每天
第四个 * 表示每月
第五个 * 表示每周的每一天
