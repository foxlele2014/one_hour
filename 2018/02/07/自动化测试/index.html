组件测试主要可以测试：

  • 交互行为（点击、输入等）
  • 状态管理和更新
  • 生命周期方法
  • 异步操作（数据加载等）
  • 条件渲染逻辑
  • 样式和类名
  • 组件间通信
  • 错误边界处理
  • 性能优化（如memo、useMemo等）

其他

每个框架的特点：
Jest: React生态，配置简单，速度快
Mocha: 灵活性高，可扩展性强
Cypress: 专注E2E测试，可视化界面
Vitest: Vite项目首选，兼容Jest API
Karma: 真实浏览器测试，支持多浏览器
选择建议：

  1. React项目 -> Jest
    Vue3+Vite项目 -> Vitest
    需要E2E测试 -> Cypress
    Node.js项目 -> Mocha
    浏览器兼容性测试 -> Karma
